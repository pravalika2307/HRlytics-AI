# HRlytics AI

## AI-Powered HR Cost Intelligence Engine

HRlytics AI is an intelligent workforce analytics platform that transforms meeting activity into actionable business insights. By leveraging AI-powered project attribution, calendar intelligence, and cost analytics, the platform helps organizations understand where employee time is spent and how workforce efforts translate into project costs.

The system automatically analyzes meetings, attributes them to projects using AI, calculates workforce expenditure, and generates real-time insights for HR, Finance, and Project Management teams.

## Problem Statement

Organizations invest significant time and resources in meetings, but often lack visibility into how employee effort is distributed across projects and how meeting activity impacts overall HR expenditure.

HRlytics AI addresses this challenge by connecting meeting data, project attribution, and workforce cost analytics into a single intelligence platform.
